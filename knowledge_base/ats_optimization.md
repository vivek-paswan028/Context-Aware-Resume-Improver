# ATS (Applicant Tracking System) Optimization Guide

## What is ATS?

Applicant Tracking Systems are software applications used by employers to manage recruitment processes. They scan, parse, and rank resumes before human review.

**Common ATS Platforms:**
- Workday
- Taleo
- Greenhouse
- Lever
- iCIMS
- SAP SuccessFactors
- BambooHR

## ATS-Friendly Formatting

### Document Structure
- Use standard section headers: "Experience", "Education", "Skills"
- Avoid headers/footers for critical information
- Use simple bullet points (•) not custom symbols
- Left-align all content
- Use standard fonts: Arial, Calibri, Helvetica, Times New Roman

### What to Avoid
- Tables and text boxes
- Graphics, images, or logos
- Columns (especially in older ATS)
- Icons or special characters
- Text inside images
- Uncommon file formats (use PDF or DOCX)

### File Format
- PDF is generally safe for modern ATS
- DOCX is universally compatible
- Never submit .pages, .odt, or image files

## Keyword Optimization

### How ATS Scoring Works
1. **Keyword Matching**: ATS scans for job-relevant keywords
2. **Context Analysis**: Checks how keywords are used
3. **Frequency**: Notes repetition (but avoid keyword stuffing)
4. **Recency**: Weights recent experience more heavily

### Keyword Strategy
- Extract keywords from the job description
- Include both full terms and acronyms (e.g., "Machine Learning (ML)")
- Use exact phrasing from job posting when accurate
- Include variations of key terms
- Place important keywords in context of achievements

### Skill Categories to Include
- **Technical Skills**: Programming languages, frameworks, tools
- **Soft Skills**: Leadership, communication, problem-solving
- **Domain Knowledge**: Industry-specific terminology
- **Certifications**: Relevant professional credentials
- **Education**: Degree names, relevant coursework

## Section-Specific Optimization

### Contact Information
- Place at top of resume
- Include: Name, Phone, Email, LinkedIn, Location (City, State)
- Use standard labels: "Phone", "Email", "LinkedIn"

### Professional Summary
- Include 3-5 key skills from job description
- Mention years of experience
- State target role or expertise area
- Keep to 2-4 lines

### Experience Section
- Use standard job titles (avoid internal titles)
- Include company name, location, dates
- Start bullets with action verbs containing keywords
- Quantify achievements with metrics

### Skills Section
- Group related skills logically
- Use commas or bullet points
- Include proficiency levels if space allows
- Prioritize skills from job description

## ATS Testing Tips

### Before Submitting
1. Copy-paste resume into plain text - check readability
2. Verify all sections parse correctly
3. Check for special characters that might break parsing
4. Ensure contact info is in body, not header/footer

### Common Parsing Issues
- Phone numbers in non-standard formats
- Email addresses with unusual formatting
- Dates in non-standard formats (use MM/YYYY or Month YYYY)
- Creative section titles that ATS doesn't recognize

## Keyword Density Best Practices

### Do's
- Include primary skills 2-3 times throughout resume
- Use synonyms and related terms
- Integrate keywords naturally into bullet points
- Match terminology from job description

### Don'ts
- Don't stuff keywords unnaturally
- Don't hide text (white text on white background)
- Don't list skills without context
- Don't exceed 5-7% keyword density

## Industry-Specific ATS Tips

### Tech Industry
- Include specific technology names (not just categories)
- List programming languages with proficiency context
- Mention development methodologies (Agile, Scrum, DevOps)
- Include relevant frameworks and libraries

### Healthcare
- Include certifications and licenses prominently
- Use standard medical terminology
- List EMR/EHR systems by name
- Include regulatory compliance experience

### Finance
- Highlight regulatory knowledge (SOX, GDPR, etc.)
- Include specific financial systems and tools
- Mention relevant licenses (CFA, CPA, Series)
- Quantify financial impact in achievements
